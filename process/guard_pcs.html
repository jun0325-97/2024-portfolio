01 Web publishing
대통령 경호처

Overview

‘대통령 경호처’ 는 와이드한 이미지, 비디오, 스크롤 애니메이션을 주로
사용한 사이트입니다. 전체적인 색감은 모노톤이며 디자인은
간결하지만 다양한 동적 효과들이 더해져 사용자에게 보다
효과적으로 정보를 전달하므로 선정하여 제작하였습니다.

Process

메인 비디오의 재생바와 재생/일시정지 버튼을 제이쿼리로 구현하였습니다.
재생/일시정지 버튼은 play() 메소드와 pause() 메소드를 사용했습니다.
비디오 재생바 기능은 간단하게 표현할 수 있도록 setInterval 을 이용해
현재 재생률 [( 100 / 비디오 전체시간) * 비디오 현재시간] 을
0.1초마다 재생바 ‘width’ 값에 ‘%’ 단위로 입력하였습니다.

addClass(), removeClass(), slide() 메소드를 이용하여
헤더의 다양한 동적 효과를 만들었습니다.
또한 콜백함수를 이용해 순차적으로
부드럽게 변하는 애니메이션을 구현하였습니다.
스크롤 값을 감지해 헤더 숨김 처리를 통해
스크롤 업/다운에 맞춰 헤더를 사라지게 하였습니다.

트윈맥스와 스크롤매직 라이브러리를 이용하여
효율적인 스크롤 애니메이션을 구현하였습니다.
원하는 시작 지점과 애니메이션이 진행될 스크롤 값을
설정하여 현재 스크롤 값에 맞추어 배경이미지를
확대하고 텍스트가 강조되게 만들었습니다.

Epillogue

사용자들이 쉽고 편리하게 접근할 수 있으면서도 디자인까지 조화로운 사이트를 만들어 보면
제작과정에서 많은 도움이 될 수 있겠다 생각하여 '대통령 경호처'를 퍼블리싱 해보았습니다.
레이아웃이 복잡하지 않아 보기엔 심플했지만 쾌적한 사용을 제공하는 동적효과가 많이 사용된 사이트였습니다.

작업할 때 막히거나 오류가 생겼을 때는 교수님이나 동기들의 도움보다는
먼저 순차적으로 해결해보거나 서칭을 이용해서 혼자 해결하려 노력했습니다.
물론 저만의 방식으로 코드를 쓰고 플러그인을 사용하려니 생각보다 작업시간을 많이 소모하기도 했습니다.
앞으로 꾸준히 서칭하고 공부해서 적재적소에만 플러그인 또한 내 것으로 만들고 효율적인 작업을 해야겠다고 느꼈습니다.
