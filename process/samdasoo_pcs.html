02 Web publishing
삼다수

Overview

‘삼다수’ 사이트는 비주얼적인 부분과 브랜드 아이덴티티를 중점으로
만들어졌습니다. 전체적인 색감은 브랜드 컬러를 살렸고
이미지에 입체적인 동적 효과들을 적용시켜 사용자에게
본인들의 브랜드를 확실히 각인시키므로 선정하여 제작하였습니다.

Process

메인 비쥬얼에 상품 이미지와 텍스트를 이용한
스크롤 애니메이션을 scrollTop() 값과 if문을 통해 구현하였습니다.
const로 각 섹션의 변하지 않는 거리값을 정의하여
스크롤 값에 맞는 애니메이션을 작동하게 하였습니다.
현재 스크롤 값과 섹션 스크롤 값의 합을 계산하여 상품 이미지의
opacity값에 입력하여 스크롤 페이드 효과를 주었습니다.

서브 페이지 메인 비쥬얼을 스크롤 할 시
현재 스크롤 값을 감지하여 메인 비주얼 배경 이미지의
position-y 값을 스크롤 값의 0.4배 값이 되도록 하여 더욱
입체적인 스크롤 애니메이션을 구현하였습니다.

모바일 사용자를 고려하여 미디어 쿼리를
이용한 반응형 웹페이지로 제작하였습니다.
스타일 시트 작성시 너비 값을 % 단위로 설정하여
보다 빠르고 효율적인 모바일 페이지를 만들었습니다.

제이쿼리로 슬라이드를 구현할 때 기본적인 동작들은
상용할 수 있게 작성하였고 모바일과 pc의 값이 다른 동작은
body의 width값을 이용한 if문으로 구분해 만들었습니다.

Epillogue

간결하지만 독특한 디자인과 입체적인 애니메이션을 많이 활용한 사이트인
'삼다수'를 보고 제작하며 다양한 아이디어를 얻을 수 있었습니다.
고퀄리티의 이미지를 사용하면서 공간감까지 더해주니 더욱 몰입감을 느낄 수 있다는 것을 느꼈습니다.

지정된 스크롤 값과 현재 스크롤 값을 계산하여 opacity를 주거나
배경 이미지를 스크롤 값의 0.4배속으로 올라가게 하는 간단한 계산식들을 쓰면서
논리적으로 접근하게 되면 코드를 더 간결하고 효율적이게 쓸 수 있다는 배움을 얻었습니다.
아직 많이 부족하지만 그럴수록 코드들을 이해하고 넘어가야겠다는 다짐이 들었습니다.
